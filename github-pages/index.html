Usf-Edge

Next-Generation Edge Proxy Management Panel

A powerful, self-hosted proxy management panel running entirely on Cloudflare Workers. Manage users with volume and time limits, track real-time traffic, and deliver beautiful subscription pages — all from the edge, with zero servers.

Features

Everything You Need

Built for performance and simplicity. Deploy in seconds, manage with ease.

Multi-User Management

Create and manage users with granular volume limits, time-based expiration, and concurrent connection caps. Full control from a clean admin dashboard.

Beautiful Subscription Pages

Polished user pages with volume usage bars, QR codes for instant client setup, step-by-step client guides, and one-click config copying.

Smart Traffic Accounting

Real-time upload and download tracking per user with D1-powered persistence. Know exactly how much data each user has consumed at any moment.

Stealth Mode

Disguises itself as a default nginx page to casual visitors. The admin panel is hidden behind a secret path, keeping your setup low-profile and secure.

Getting Started

Prepare Your API Token

Follow these steps to generate a Cloudflare API token with the required permissions.

1

Create a Cloudflare Account

If you don't already have one, sign up for a free Cloudflare account. Workers and D1 are included in the free tier.

dash.cloudflare.com/sign-up →

2

Get Your API Token

Generate a token with the exact permissions needed for deployment:

  • Go to Cloudflare Dashboard → My Profile → API Tokens
  • Click "Create Token"
  • Use the "Edit Cloudflare Workers" template
  • Verify the token includes these permissions:
  • Account > Workers Scripts > Edit
  • Account > D1 > Edit
  • Account > Workers KV Storage > Edit
  • Click "Continue to summary" then "Create Token"
  • Copy the token — you'll paste it in the deploy tool below
3

Deploy Using the Tool Below

Scroll down to the One-Click Deploy section, paste your token, and hit Deploy. Everything else is handled automatically.
